==============
repo-templates
==============
Repository templates.

repo set-up

needed packages: ``git``, ``pre-commit``

optionally create a virtual environment

run:::

  $ pre-commit install
  $ pre-commit autoupdate
  $ pip install pylint isort

LICENSE.txt

.pre-commit-config.yaml
.pylintrc
README.rst
requirements.txt
setup.cfg
setup.py
tox.ini
.travis.yml
